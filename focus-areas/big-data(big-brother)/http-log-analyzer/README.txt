Please follow below instructions to run the sample

* Unzip BAM 2.4.0
* Replace the file, access.log in <BAM_HOME>/samples/httpd-logs/resources/ from the http log file downloaded from http://ita.ee.lbl.gov/html/contrib/EPA-HTTP.html .
* Start the BAM server.
* Deploy the modified HTTPD_Logs.tbox given in this git location.
* Go to <BAM_HOME>/samples/httpd-logs and run the sample to publish log events.
* Once the hive scripts are executed, all summarised data will be available in h2 database.
* Generate gadgets as described in the webinar to visualize results.

You will be able so see analyzed log data in the WSO2 BAM Gadget Portal and will get insights of the log data.
